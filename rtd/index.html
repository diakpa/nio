Install Instructions

custom camgirl.cloud profile designs

Camgirl.Cloud allows registered members to add custom HTML markup to customize their bio. Options available are HTML and CSS boxes but HTML and inline CSS markup can be used.

To access this settings you need to be loggedin on Camgirl.Cloud and navigate to Profile Settings
Once on Profile scroll until you see 2 boxes named Home Page (HTML) and Home Page (CSS).

Paste your custom HTML markup code or HTML with inline CSS in the first box (top one, named HTML)

Paste your custom CSS markup code in the lower box named CSS - skip this step if you are not using custom css stylesheet

To edit the content and customize designs made by me, you could use any HTML editor/ WYSIWYG editor or in a plain text editor (harder to edit without coding skills). But for a small fee, I will help you!

  1. Download your favorite design
  2. Open provided markup codes in your favorite HTML or Text editor
  3. Make sure all your replacement images are hosted online and replace them inside the source markup code
  4. Add/change/edit texts, links, buttons
  5. Make sure you've closed all markup tags and your editing didn't messedup overall design
  6. Go ahead with install process
